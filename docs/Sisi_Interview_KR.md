*Created: 2026년 5월 13일*
*한국 친구·지인 인터뷰용 + MVP 사용자 테스트 protocol*

---

## 🎯 두 가지 인터뷰 형식

### A. **발견 인터뷰** (Discovery — 30-40분)
사용자 manifestation 경험·needs·pain points 깊이 이해.

### B. **사용자 테스트** (Usability — 20-30분)
본인 MVP (localhost:3000/app) 직접 사용시키고 reaction 관찰.

**둘 같이 가능:** 발견 인터뷰 → 그 다음 MVP 보여주기. 총 50-60분.

## 📋 발견 인터뷰 스크립트 (30-40분)

### Opening (2분) — 분위기 셋팅

```
안녕 [이름]! 시간 내줘서 진짜 고마워.

빠르게 컨텍스트:
내가 Sísí 라는 manifestation 영적 friend 앱 만들고 있어. 
출시 전에 7명 깊이 인터뷰 중이야.

이거 영업 통화 아니야 — 진짜로 너가 어떻게 manifestation 
하는지 배우고 싶어. 정답 없어. 솔직하게 다 답해줘.

녹음해도 될까? 듣기에 집중하려고. 내 노트용이야.

시작해도 돼?
```

### Section 1: 배경 (8분)

**핵심 질문:**

1. *"Manifestation 어떻게 처음 알게 됐어? 언제부터 관심 가졌어?"*
   - 흘려서 들어. 그들의 *첫 만남* 보면 motivation·trigger 알 수 있음.

2. *"지금 너의 manifestation 실천을 한 줄로 설명하면? 일주일에 어떻게 보내?"*
   - 빈도 파악 (매일·매주·가끔?)

3. *"너한테 manifestation이 진짜 의미하는 게 뭐야?"*
   - 그들의 philosophy 캐치 (Goddard? Hicks? Lacy Phillips? casual?)

**관찰 포인트:**
- 어떤 단어 자주 써? ("끌어당김", "주파수", "alignment", "trust" 등)
- 영성 vs 신경과학 vs tech 어느 결?
- *"진짜 믿음"* vs *"playful 믿음"* 어디?

### Section 2: 도구 & 방법 (10분)

4. *"지금 manifestation 위해 어떤 앱·도구·방법 써?"*
   - 답한 거 다 적기 (저널·앱·vision board·affirmation 카드 등)

5. *"제일 좋아하는 [그 앱·방법] 의 뭐가 좋아?"*
   - 그들이 *love* 하는 부분

6. *"그게 뭐가 답답해? 짜증나는 게 있어?"*
   - 진짜 pain points

7. *"비전 보드 있어? 어떻게 관리해?"*
   - 본인 핵심 기능 검증

8. *"manifestation 한 거 트래킹해본 적 있어? — 뭐를 끌어당겼는지?"*
   - Synchronicity Timeline 검증 (보통 *"안 해. 근데 좋겠다"* 답할 거)

9. *"지금 하는 거랑 진짜 필요한 거 사이에 갭이 뭐야?"*
   - **이 답이 골드.** 진짜 needs

### Section 3: Pain Points 깊이 (8분)

10. *"manifestation 실천 그만둔 적 있어? 왜?"*
    - 본인 *"streak 압박 X"* 포지셔닝 검증

11. *"manifestation 앱들 보면서 *손발 오그라드는* 부분 있어?*
    - 본인 anti-cringy 포지셔닝 검증

12. *"네 주변에 manifestation 회의적인 사람 있어? 뭐라고 말해?"*
    - 회의주의자 진입 모먼트

13. *"manifestation 좋아하지만 *too woo*는 거부감 있어?"*
    - 본인 brand 정확히 검증

14. *"manifestation에 대해 누군가 앞에서 말하는 거 부끄러워?"*
    - 진짜 emotional layer

### Section 4: Pricing & Beta (5분)

15. *"이런 앱이 진짜 잘 만들어졌으면, 한 달에 얼마까지 낼 거 같아?"*
    - 5천원? 1만원? 1.5만원?

16. *"매일 Sísí 음성 메시지, 무제한 synchronicity timeline, AI vision board, 이런 게 premium 기능이면 낼 의향 있어?"*

17. *"내가 60일 안에 베타 만들면 테스트해볼래?"*

18. *"2-4주 후에 prototype 보여줄 때 연락해도 돼?"*

### Closing (2분)
```
정말 도움됐어, 진심으로.

마지막 몇 가지:
1. 너랑 비슷한 manifestation 좋아하는 사람 또 있어?
2. 베타 가장 먼저 보내줄게 — 60일 후.
3. 우리가 안 다룬 거 중에 manifestation에 대해 추가하고 싶은 게 있어?
```

## 🖥️ B. MVP 사용자 테스트 Protocol (20-30분)

### 셋업

본인 두 옵션:
- **A. 친구가 본인 집에 와서 본인 노트북으로 테스트** (가장 정확)
- **B. 줌으로 화면 공유** — 친구가 본인 컴퓨터에서 못 함, 본인이 친구 카메라 보면서 진행
- **C. 임시 Vercel deploy** — vercel에서 preview URL 생성 → 친구 폰으로 직접 접속

지금 단계에선 **A 또는 B** 추천 (MVP 아직 localhost).

### 테스트 진행 (15-20분)

**전제: 친구한테 *맞고 틀린 답 없다, 그냥 자연스럽게 반응해줘* 강조**

```
"내가 만든 앱 보여줄게. 평소처럼 자연스럽게 써봐. 
*맞고 틀린 답 없어* — 헷갈리면 헷갈린다고 말해, 좋으면 좋다고 말해. 
머릿속에서 *생각하는 거 다 소리내서 말해줘.* 
(이거 'think aloud' 라고 해.)

3가지 해볼 거야:
1. 앱 처음 사용 (가입·온보딩)
2. 첫 vision board 만들기
3. 사용 후 느낌

녹화해도 될까?"
```

### Task 1: 첫 가입·온보딩 (5분)
브라우저 열어서 localhost:3000 보여줌 → *"마음대로 써봐"*

관찰 포인트:
- 어디서 멈춰? 헷갈려?
- 어떤 단어·instruction 안 통해?
- 본능적으로 어디 클릭해? (헷갈리면 어디 가려고 해?)
- 표정·반응 (오 신기·아 답답·헷갈림)
- 본인 카피·voice 어떻게 felt? (*"morning. what is calling you in today?"* 결)

질문 (Task 후):
- *"방금 그거 어땠어?"*
- *"어디서 가장 헷갈렸어?"*
- *"앱이 너한테 뭐라고 말하는 것 같아?"*

### Task 2: Vision Board 만들기 (10분)

만약 Claude Code가 이미 완성했으면 사용자한테 *"이제 첫 vision board 만들어봐"* 시키기.

관찰 포인트:
- "what is calling you in today?" 에 뭐라고 입력? (이게 manifestation pattern)
- 4개 이미지 받았을 때 reaction (감탄·실망·놀라움)
- 어떤 이미지 고르나? 왜?
- "this is the one" 누른 후 뭐 기대했어?

질문 (Task 후):
- *"이미지 너랑 얼마나 맞아 (1-10)?"*
- *"이거 wallpaper로 진짜 쓰고 싶어? 왜·왜 X?"*
- *"이미지 보고 뭐 느껴졌어?"*

### Task 3: 전체 reaction (5분)

질문:
- *"이 앱 한 줄로 친구한테 설명하면 어떻게 말할 거 같아?"*
- *"1-10점 — 한 달에 얼마면 사용할 가치 있어?"*
- *"베타 사용자로 60일 후 또 써볼래?"*
- *"가장 좋았던 부분? 가장 답답했던 부분?"*

### Closing
```
정말 도움됐어. 베타 가장 먼저 보내줄게.
주변에 manifestation 좋아하는 사람 또 있어?
```

## 🎯 Synthesis — 5-7명 끝나면

### 5가지 가설 검증 (한국 사용자 답 기준)

| 가설 | YES (몇 명) | NO (몇 명) | 패턴 |
|---|---|---|---|
| 1. cringy manifestation 앱 거부감 있다 | _/7 | _/7 | |
| 2. Vision board on lock screen 가치 있다 | _/7 | _/7 | |
| 3. Synchronicity timeline 원한다 | _/7 | _/7 | |
| 4. "친구" 톤 felt 좋다 (vs 신탁) | _/7 | _/7 | |
| 5. 월 1.2만원 (≈$9.99) 낼 의향 | _/7 | _/7 | |

### 패턴 질문
1. **5명 이상이 같은 단어 자주 쓴 거?** → Voice library 추가
2. **5명 이상이 같은 pain point 말한 거?** → MVP 우선 기능
3. **5명 이상이 같은 앱 칭찬한 거?** → borrow할 거
4. **본인이 가장 놀란 인사이트?** → Brand·MVP 반영
5. **본인 가설을 *반박* 한 거?** → 정직하게 pivot

### 한국 시장 specific 인사이트
- **결제 의향**: 한국은 미국보다 월 구독 거부감 있음. *연 결제* 또는 *1회 결제* 선호?
- **언어**: 한국어 voice 옵션 원할까? (ElevenLabs 한국어 voice도 가능)
- **시즌성**: 신년 (1-2월), 생일·새로운 시작 때 manifestation peak

## 💎 솔직한 팁

1. **첫 1-2명은 가장 가까운 사람** (엄마·자매·친한 친구). 본인 brand·voice 익숙해진 후 다른 사람.

2. **사용자 테스트할 때 본인이 옆에서 *조용히 관찰만*.** 친구가 헷갈려해도 *"여기 클릭해!"* 도와주면 안 됨. 본인이 도와줘야 알면 = MVP가 unclear한 거. 그게 학습.

3. **친구가 *"좋아 좋아"* 만 반복하면 압박감 느끼는 거.** *"솔직히 별로면 별로라고 말해도 우정 안 깨져"* 농담식으로 강조.

4. **5명 이상이 같은 답 = 진짜 신호.** 1-2명은 우연.

5. **인터뷰 후 30분 안에 노트 정리 무조건.** 안 하면 까먹음.

6. **녹음하면 후에 다시 들으면서 놓친 인사이트 발견.**
