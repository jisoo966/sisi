**규칙 reminder:**
- 항상 lowercase
- 줄임말 X (you, your — NOT u, ur)
- Teen interjection X (yeah, ok so, bestie)
- Mature wise friend 톤
- 이모지 극히 절제 (🌙은 밤 인사에만)
- Self-aware spirituality OK
